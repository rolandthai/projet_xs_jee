# projet_xs_jee
projet version jee

### Required configuration ###
* JDK 1.7 or above
* Maven 3
* Git
* Tomcat 7 or above

## Running  ##

### Compilation ###
```
mvn clean install
```
This command line creates the executable file **war** in the /target directory.

### Running ###
put the war file in tomcat server

run as :
```
http://[tomcat server]/xspeeditjee/oui
```
